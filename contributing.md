# Contributing

Pull requests are welcome. This list is for AI products built by independent
founders and small teams.

## Adding a Startup

Add the entry under the most relevant category in `README.md` using this format:

```md
- [Name](https://product.url) - One-line tagline.
```

## Inclusion Criteria

- Bootstrapped, pre-seed, or angel-funded only.
- No Series A+ companies, big-tech subsidiaries, or big-tech product pages.
- The startup must offer a product an end user can sign up for, download, or use.
- The product must use AI, machine learning, LLMs, or agents in a meaningful way.
- The website must be live and not a parked domain.

## Quality Bar

- Keep descriptions short, factual, and sentence-cased.
- Link directly to the product or project page.
- Avoid affiliate links, tracking parameters, and launch-only URLs when a canonical product URL exists.
- Do not add duplicate entries.
